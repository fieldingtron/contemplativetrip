# AGENTS

This repository uses the project name: contemplativetrip.

## Purpose
- Next.js + TinaCMS site for contemplative retreat content.
- Content lives in local MDX files under content/articles, content/events, and content/direction.

## Working Rules
- Use pnpm for package management.
- Keep changes small and scoped.
- Prefer documentation and metadata updates unless explicitly asked to change UI or app behavior.

## Common Commands
- Install: pnpm install
- Dev: pnpm run dev
- Build: pnpm run build
- Start: pnpm run start
- Lint: pnpm run lint
